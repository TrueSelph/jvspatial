# Database Examples

This directory contains examples demonstrating database features and customization:

- `custom_database_example.py` - How to create custom database backends
- `custom_database_registry_example.py` - Database backend registration system
- `database_switching_example.py` - Dynamic database switching
- `modern_query_interface.py` - Modern query builder interface
- `orm_demo.py` - Object-Relational Mapping features
- `unified_query_interface_example.py` - Unified query interface across backends
